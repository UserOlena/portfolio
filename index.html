Global Weather Forecast

The Global Weather Forecast application is a valuable tool that provides users with current and 5-day weather forecasts for locations worldwide. The application makes use of the openweathermap.com API, which supplies accurate and real-time weather data, including temperature, wind speed, humidity, and weather icons that indicate the current weather condition. By utilizing this information, users can make informed decisions and plan their activities accordingly. The web application also includes a feature that stores the names of previously searched cities in the user's browser local storage. Up to 10 objects containing data for each city are saved, with the most recent search displayed at the top of the history buttons. The user is able to easily access this history and delete it if desired. If the user chooses to delete the history, all data associated with the previously searched cities will be removed from both the local storage and the page.

Password Generator

The Password generator web application was created to produce secure passwords for staff members. Once "generate password" button is clicked, the user is prompted to select which types of characters they would like to include in their password, such as uppercase and lowercase letters, numbers, or special characters. Additionally, the user must specify the desired password length within the range of 8 to 128 characters. The password generator guarantees that at least one symbol of each character type is included in the password and then shuffles it again for enhanced security before presenting it to the user. To improve the user experience, the password generator features a "copy" button, allowing users to effortlessly copy the password to the clipboard.

Coding Quiz Challenge

The Code Quiz web application offers users a dropdown menu to select from three coding subjects: HTML, JavaScript, and CSS. Each subject has six questions, and for each question, the user is presented with four answer options in a button format. Once the user chooses a subject and clicks the start button, a 60-second countdown timer begins. If the user selects an answer button, the background color and the chosen button element will change to red if the answer is incorrect, or green if it is correct. If the user selects a wrong answer, 10 seconds will be deducted from the remaining time. The next set of questions and answers automatically loads when the user selects an answer button. The quiz ends when the user answers all questions within the time limit or when the timer reaches zero. At the end of the quiz, the user receives their result as a percentage, with 100% being the highest score achievable. Users are asked to enter their initials to save their result on the High Score Board. If the user provides their initials, the quiz result is saved to Local Storage. The High Score Board is accessible at any time, even after page reload, unless the user clicks the Clear High Scores button, which deletes all Quiz Results data from Local Storage.
